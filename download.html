Prerequisite

You need a Java Development Kit (JDK) at least in version 21 installed to be able to run JSkat on your computer. We recommend to always have the latest Long Term Support (LTS) release installed. JSkat is developed with the JDKs from Adoptium.org.

Latest stable version

Until now, we don't have released a functionally complete and stable version of JSkat, so it still has beta status. But we will continue to release snapshots of our work in progress. You can also get the latest version of the development version from our Git repository at GitHub.

Latest release

The latest release is version 0.23.0 that was released in March 2024. You can get it on the website of our GitHub project:

https://github.com/b0n541/jskat

There is also a PAD file you can observe to get information about a new release.

Git snapshot

If you are more interested in the development progress or if you want the lastest builds you can always get a snapshot by anonymous Git access:

https://github.com/b0n541/jskat.git
