Latest stable version

Until now we don't have released a functionally complete and stable version of JSkat, so it still has beta status. But we will continue to release snapshots of our work in progress on a regular basis (every three to four months) and you can also get the latest version of the development version from our SVN at sourgeforge.org.

Latest release

The latest release is version 0.6 that was released in August 2007. You can get it on the website of our sourgeforge project:

http://sourceforge.net/projects/jskat/

SVN snapshot

If you are more interested in the development progress or if you want the lastest builds you can always get a snapshot by anonymous SVN access:

svn co https://jskat.svn.sourceforge.net/svnroot/jskat jskat/trunk
