Latest stable version

Until now we don't have released a functionally complete and stable version of JSkat, so it still has beta status. But we will continue to release snapshots of our work in progress on a regular basis (every three to four months) and you can also get the latest version of the development version from our SVN at sourgeforge.org.

Latest release

The latest release is version 0.16.1 that was released in December 2017. You can get it on the website of our GitHub project:

https://github.com/b0n541/jskat-multimodule

There is also a PAD file you can observe to get informations about a new release.

Git snapshot

If you are more interested in the development progress or if you want the lastest builds you can always get a snapshot by anonymous Git access:

https://github.com/b0n541/jskat-multimodule.git
