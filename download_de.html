Letzte stabile Version

Bis jetzt haben wir noch keine vollständige funktionierende und stabile Version von JSkat freigegeben. Deshalb hat JSkat auch noch den Beta-Status. Wir werden aber in regelmäßigen Abständen Ergebnisse unserer Arbeit herausbringen. Sie können entweder die letzte Version von JSkat herunterladen oder eine Entwickler-Version direkt aus unserem Subversion-Repository auf sourceforg.org bauen.

Letzte Version

Die letzte Version ist Version 0.11, die im August 2012 freigegeben wurde. Sie können diese Version von der Website unseres Sourceforge-Projektes herunterladen:

http://sourceforge.net/projects/jskat/

Es gibt auch eine PAD-Datei, die Sie überwachen können, um über neue Versionen informiert zu werden.

SVN-Repository

Wenn Sie mehr am Fortschritt der Entwicklung von JSkat interessiert sind oder immer die neueste Version vor allen anderen testen wollen, können Sie diese jederzeit per anonymen SVN-Zugriff erhalten:

svn checkout svn://svn.code.sf.net/p/jskat/code/jskat/trunk
